# Manual Testing Project – Login Page

## Project Overview

This project contains manual testing documentation for a demo login page.

The goal of this project is to practice writing test cases, executing manual tests, documenting actual results, and preparing QA documentation as part of a Software Testing portfolio.

## Application Under Test

**Application:** The Internet – Login Page
**Feature tested:** Login functionality
**Testing type:** Manual Testing

## Test Scope

The following login scenarios were tested:

* Login with valid credentials
* Login with invalid username
* Login with invalid password
* Login with empty username and password fields
* Logout after successful login

## Test Types Used

* Functional Testing
* Negative Testing
* UI Testing
* Usability Observation
